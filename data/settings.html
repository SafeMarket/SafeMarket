Master Private Key (xprv)

If you have Electrum 2.x installed, you can have electrum generate a master private key for you. First, create a wallet seperate from your personal transactions then go to the console tab.

  • If your wallet is password protected, run wallet.get_master_private_key('x/', gui.password_dialog())
  • If your wallet is NOT password protected, run wallet.get_master_private_key('x/', None)

Your master private key should be between quotes, starting with 'xprv'.
